Short follow-on to part 1 of the Perentie chassis assembly. ## Steps

### 1. Install the bash guard under the motor

With the vehicle up on the hoist, install the bash guard: the large plate that mounts underneath the motor. This has its own dedicated process document, follow that for the full fitment procedure.

![Step 1](https://raw.githubusercontent.com/davebudge/process-documents/main/output/perentie-chassis-component-assembly-part-2/frames/step-01.jpg)

### 2. Install the gearbox mount crossbar

Install the crossbar that replaces the old gearbox mount. It bolts straight in. Each side is held by two M8 fasteners (M8 x 25mm `VERIFY`), one at each location, that pass through and thread into the rivnuts already fitted to the mount.

![Step 2](https://raw.githubusercontent.com/davebudge/process-documents/main/output/perentie-chassis-component-assembly-part-2/frames/step-02.jpg)
